# 文明6术语网页

这个文件夹是一个独立的本地网页项目，用来查询《文明6》文明百科右侧数值面板。

## 最终文件

- `index.html`：最终网页，直接用浏览器打开即可，也可以发布到 GitHub Pages。
- `assets/civ6-data.js`：从 civilopedia.net 中文文明百科提取的数值面板数据。
- `assets/civ6-hero.png`：网页顶部横幅图。
- `tools/`：数据抓取脚本，用于重新生成或扩展数据。

## 使用方式

双击 `index.html`，或者把它拖进浏览器。网页支持按分类筛选，也支持搜索条目名称、相邻加成、生产力消耗、伟人点数、放置条件等数值字段。

## 是否可以删除

如果不再需要这个网页，可以直接删除整个 `文明6术语网页` 文件夹。不要只删除 `assets`，否则网页顶部图片会丢失。

## 素材说明

顶部横幅是为这个网页生成的策略地图风格图片，不使用游戏截图或官方 Logo。数值数据来自 civilopedia.net 的中文《文明百科》页面，网页只展示右侧数值面板，不摘录中间的背景介绍正文。
